工作第一年总结与下一年工作展望

于彬 2020-07-03 总结 工作 成长 不足

# 个人总结

时间过的很快，一晃我已经从一名初出校园的毕业生变成有一年工作经验的职场人了。这一年里，很漫长，但有短暂。失意的时候，时间变得难熬。开心时刻也一眨眼就过去了。不唧唧歪歪了，我写这篇文章主要是记录自己这一年的收获与变化，同时期待一下明年更好的自己！

# 2019-2020

我将从以下几个方面对自己过去的一年进行总结：自我分析，学习进程，生活变化等几个方面。不过在这之前我要用一个关键字概述一下这一整年。

# 关键字

自打我有写这篇总结的想法时，我就一直在寻找恰当的关键字形容这一年。最终我认为这个关键字应该是 摸索 吧。职场新人对职场规则的摸索，离开校园后生活习惯的摸索，个人学习方法的摸索，......总的来说，摸索基本能够描述我第一年的工作状态吧。

# 自我分析

我认为一个人对自己有一个正确的认知是极其重要的。那说完年度关键字，就说说这一年下来，我对自己的认识与分析吧。

# 优势

  1. 大心脏

    工作这一年里，没想象的顺利。领导较低的认可度，紧张的业务压力，优秀同事间的隐形竞争，工作经验的不足，这些都曾让我失落沮丧过。可能每个职场人都会遇到各种各样不如意，毕竟人生不如意十有八九，我又有什么差别呢？不过对我来说，没有什么可以打败我，没有什么可以让我彻夜难免。每每遇到不如意，我都这样安慰自己，“于彬，你以后是要做大事的人，这点事都能把你打倒，以后怎么做得了大事！”。我挺自信的，自信到自负，也许自命不凡不是好事，但对我来说，我也确实因此跨过了很多坎。即使今天再失落，再沮丧，第二天，我依旧能自信的笑出来。

  2. 不服输

    不得不承认，身边优秀的人还是比较多的。同事绝大多数都是985的研究生。与其一起共事，这对于一个211的本科生来说，确实存在客观上的差距。为什么说是“客观”上的呢？因为主观上，我不觉得我本质上比谁谁谁差，即使领导更看好他，即使他受到更多关注，更多表扬。我承认，我现在也许做的不如一些人好，但是，这只是现在，以后的路还长。属于我的我会拿回来，没有这个劲头儿，我的人生又有什么意义？

# 不足

  1. 英语能力

    写代码，每天都要与单词打交道，虽说写代码靠的是逻辑，但是英语单词却是载体。我得承认，我缺少对英语重要性的认识，哪怕在写总结的现在，也没能意识到英语对我来说的重要程度。

  2. 认知水平

    认知决定人生高度。我觉得我的认知就是个普通人的认知水平，不存在认知方面的优势。好幸运的是，我能认识到我的认知水平有限。下一年，或者接下来的几年，我要有意识地提高自己的认知能力。这对以后人生的影响应该是意义非凡的。

  3. 思维模型

    “思维模型”一词，我第一次听到应该是在B站的“所长林超”202年4月23日的节目中。他提到的思维模型给我留下很深的印象，《穷查理宝典》应该是一块很好的敲门砖。

  4. 阅读能力

    都说“人拼到最后往往拼的都是知识储备”，“读书与不读书的差距往往体现在30岁之后”。我理性的想觉得是有道理的，但仍将信将疑，可能是我的认知能力不够吧。提高自己的知识面是必要的。

# 生活变化

生活上的变化主要体现在两个方面吧：作息以及体重。

# 生活作息

由之前大学的1点多睡，10点醒。现在变成了12点睡，8点醒。（即使周末也不睡太晚）。个人觉得比较欣慰，夸夸自己，希望自己再接再厉保持下去，做一个白天精神充沛的小伙子。

# 体重

得益于2020年5，6月份运动了两个月，体重从之前非常稳定的140斤降到了135斤，最主要的，啤酒肚基本没了。就不多表达喜悦之情，总之2020也要再接再厉。毕竟也快要找女朋友了。
